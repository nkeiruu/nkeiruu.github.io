Automating the collection of search result data with Custom Search JSON API

Article by Nkeiru Campbell-Ubadike.

Photo by Benjamin Dada on Unsplash

I wanted to collect links and descriptions about online content related to linear accelerators (and 22 particle accelerator related terms) for a work project. Rather than search each term individually and copy-paste URLs and description for the top search results, I used Custom Search JSON API for quickly and easily collecting search results. So I made a for a list of particle accelerator related terms and fed each one through my short program.

If I were to improve this, I would rewrite the program so that it accepts a list of terms as file (txt or csv). Result: a neat little database of titles, links and short summaries for 100 search results for a given search term. And well rested ctrl + c or ctr +v fingers and saved for more important matters.
